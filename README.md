# Portfolio BUT Réseaux & Télécommunications

Bienvenue sur le dépôt de mon portfolio réalisé dans le cadre de mon **BUT Réseaux & Télécommunications** (1ère année).

Ce projet a pour but de présenter mon parcours, mes compétences acquises en formation et mes projets personnels et académiques.

## 👤 À propos de moi

Je suis **Yanis Durel**, étudiant passionné par l'infrastructure réseau, le développement et la cybersécurité. Mon objectif est d'évoluer vers l'ingénierie réseaux et l'administration de systèmes sécurisés.

## ✨ Caractéristiques du projet

- **Design Moderne** : Utilisation du glassmorphism, de gradients dynamiques et d'animations fluides pour une expérience utilisateur premium.
- **Responsive Design** : Entièrement adapté aux mobiles, tablettes et ordinateurs.
- **Section Projets** : Vitrine de mes réalisations académiques (SAE) et personnelles.
- **Bilans & Compétences** : Suivi de mon apprentissage à travers les différents modules du BUT R&T.

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique du contenu.
- **CSS3 (Vanilla)** : Design system personnalisé, Flexbox, Grid, et variables CSS.
- **JavaScript** : Intégration d'animations au défilement (Scroll Reveal) et menu interactif.
- **Google Fonts** : Polices "Outfit" et "Parkinsans" pour une typographie moderne.

## 🚀 Installation locale

Pour visualiser le projet localement :

1. Clonez le dépôt :
   ```bash
   git clone [url-du-depot]
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur ou utilisez une extension type "Live Server".
