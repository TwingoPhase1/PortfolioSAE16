SAÉ 14

Se Présenter sur Internet

1. Contexte

Objectifs : Maîtriser son identité numérique, réaliser un support de communication professionnel et développer sa visibilité en ligne à travers des technologies web.

Environnement Technique : HTML, CSS, Git, Hébergement web (Github Pages), Réseaux sociaux professionnels.

Rôle : Création intégrale du site web et gestion du dépôt de versionnement.

🔗 Voir le Dépôt GitHub

2. Apprentissages Critiques (AC)

  • AC1 : Produire un support de communication numérique adapté à son projet professionnel (Portfolio).
  • AC2 : Gérer son identité numérique et maîtriser les paramètres de confidentialité et de visibilité.
  • AC3 : Utiliser des outils de versionnement décentralisés (Git/GitHub) pour suivre et publier un projet.

4. Analyse Réflexive

Les difficultés rencontrées :

Créer une identité visuelle cohérente et gérer l'hébergement continu de mon code via Git sans erreurs de merge conflictuelles.

Comment elles ont été surmontées :

En m'inspirant de projets de design existants et en m'appuyant sur les commandes de base git status et git branch fréquemment pour éviter les erreurs.

Ce que j'ai appris et adapté :

J'ai pris conscience que la présence en ligne était la première chose qu'une entreprise ou qu'un contact professionnel regarde. Savoir construire ce CV en ligne et utiliser un outil essentiel comme Git m'ouvre les portes d'approches collaboratives pour l'avenir.
